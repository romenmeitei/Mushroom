# AI-Assisted Semantic Mapping of Mushroom-Poisoning Literature

This repository contains the supporting code and curated datasets used for the manuscript:

**AI-Assisted Semantic Mapping of Global Mushroom-Poisoning Research: A BERTopic-Based Bibliometric and Toxicological Analysis**

The workflow performs automated literature retrieval, metadata harmonization, natural language processing (NLP), transformer-based semantic embedding, BERTopic modeling, expert-guided topic curation, temporal trend analysis, and statistical evaluation of global mushroom-poisoning research.

---

## Repository Overview

This repository is organized around three sequential Google Colab notebooks:

| Notebook | Purpose |
|---|---|
| `1. NLP_Data_extraction.ipynb` | Retrieves records from PubMed, Scopus, and optional Web of Science validation; harmonizes metadata; removes duplicates; constructs the NLP-ready corpus; performs initial BERTopic modeling. |
| `2. NLP_Data_analysis.ipynb` | Applies expert-guided thematic consolidation; generates final curated themes; produces final theme-level datasets and manuscript figures. |
| `3.NLP_Statistical_analysis.ipynb` | Performs statistical analyses including temporal trend modeling, chi-square tests, enrichment analysis, thematic diversity analysis, and species/toxin/country summaries. |

The repository also includes key CSV files generated by these notebooks and used in the manuscript.

## API Requirements

### PubMed

PubMed records are retrieved using the NCBI Entrez API through Biopython.

You should set your NCBI email address in the notebook:

```python
ENTREZ_EMAIL = "your_email@example.com"
```

### Scopus

Scopus retrieval requires an Elsevier Scopus API key and institutional access.

In Google Colab, store the API key using **Secrets**:

```text
SCOPUS_API_KEY
```

Then access it in the notebook as:

```python
from google.colab import userdata
SCOPUS_API_KEY = userdata.get("SCOPUS_API_KEY")
```

### Web of Science

Web of Science retrieval is optional and was used for supplementary validation. If used, store:

```text
WOS_API_KEY
```

## Notes on Expert Curation

The expert-guided semantic validation step is essential. BERTopic generates machine-derived semantic clusters, but not all clusters are biologically or toxicologically relevant to mushroom poisoning. Each topic was reviewed using:

- top c-TF-IDF topic keywords,
- representative publication titles and abstracts,
- biological relevance to mushroom poisoning,
- clinical/toxicological interpretability,
- and overlap with other themes.

Topics were assigned one of three decisions:

| Decision | Meaning |
|---|---|
| INCLUDE | Directly relevant to mushroom-poisoning research. |
| EXCLUDE | Not relevant to mushroom poisoning or dominated by unrelated biomedical content. |
| PARTIAL | Partly relevant but requiring merging, relabeling, or refinement. |
